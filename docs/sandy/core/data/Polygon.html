POLYGON_MAPproperty 
public static var POLYGON_MAP:Dictionary

This property lists all the polygons. This is a helper property since it allows all polygons of a scene to be retrieved from a single list by its unique identifier. A polygon's unique identifier can be retrieved useing myPolygon.id.


Example
The examples below shows how to retrieve a ploygon from this property. 
   var p:Polygon = Polygon.POLYGON_MAP[myPolygon.id];

get3DFrom2D()method 
public function get3DFrom2D(p_oScreenPoint:Point):Point3D

Returns a Point3D (3D position) on the polygon relative to the specified point on the 2D screen.

Parameters
p_oScreenPoint:Point

Returns
Point3D — A Point3D that corresponds to the specified point.

Example
Below is an example of how to get the 3D coordinate of the polygon under the position of the mouse: 
   var screenPoint:Point = new Point(myPolygon.container.mouseX, myPolygon.container.mouseY);
   var scenePosition:Point3D = myPolygon.get3DFrom2D(screenPoint);
